# /shot-wrighter — 슈퍼말순 스타일 동물 쇼츠 대본 생성

> 카테고리와 주제만 입력하면, 슈퍼말순 채널 스타일의 동물 쇼츠 대본을 자동 생성합니다.

## 트리거

```
/shot-wrighter "강아지 미용실" --animal 강아지
/shot-wrighter "고양이 카페 알바" --animal 고양이
/shot-wrighter "동물병원 첫 방문" --animal 강아지 --mood 긴장
```

## 입력

| 파라미터 | 필수 | 설명 | 기본값 |
|---------|------|------|--------|
| 주제/상황 | O | 쇼츠의 상황 설명 (한 줄) | - |
| `--animal` | | 주인공 동물 (강아지/고양이/햄스터/앵무새 등) | 강아지 |
| `--mood` | | 감정 분위기 (웃긴/감동/긴장/황당/귀여운) | 웃긴 |
| `--name` | | 주인공 이름 | 말순이 |
| `--setting` | | 배경 (시골/도시/학교/회사/병원/가게 등) | 시골 |

## Expert Knowledge: 슈퍼말순 채널 분석

### 채널 프로필
- **채널명**: 슈퍼말순 (구독자 2.56만+)
- **컨셉**: "할머니와 사는 시골 강아지 말순이채널"
- **핵심 가치**: "힘든 세상 속에서 잠깐 웃고 쉬어갈 수 있는 공간"
- **형식**: AI 생성 이미지 + 자막 나레이션 쇼츠 (20~57초)

### 히트작 분석 (조회수 기준)

| 제목 | 조회수 | 핵심 패턴 |
|------|--------|----------|
| "누군지는 저도몰라요" | 148만 | 반전 유머, 정체 숨기기 |

### 영상 시각 분석 (실제 스크린샷 기반)

| 영상 | 시각 요소 | 자막 스타일 |

| "인생이원래그런거에요" | AI 비숑, 노란 점퍼, 막대사탕, 거리 배경 | 상단: "인생이 뭐라고 생각하세요?" (초록 글씨), 하단: "인생은..." |
| "삼거리미자네멍품점" | AI 비숑, 빨간 리본, 옷가게 배경, 옷걸이 | 상단: "광고아님" (초록 글씨), 하단: "예쁜옷 입어야 된다고.." |

### 시각 제작 패턴 (AI 이미지 생성용)
- **주인공 외형**: 하얀 비숑 프리제 강아지 (슈퍼말순의 트레이드마크)
- **소품**: 마이크 (인터뷰/토크 형식의 상징)
- **코스튬**: 상황에 맞는 옷 (요리사모자, 점퍼, 한복 등)
- **배경**: 상황에 맞는 장소 (시장, 거리, 가게, 집, 꽃밭)
- **텍스트 레이아웃**: 상단에 큰 임팩트 텍스트(핵심 단어) + 하단에 자막
- **색상**: 상단 텍스트는 빨간색 또는 초록색(네온), 하단 자막은 흰색
- **시리즈 태그**: [빙의], [성대모사] 등 시리즈명 표시

### 슈퍼말순 대본 공식

#### 1. 캐릭터 설정
- **주인공**: 동물 (1인칭 화자, 사람처럼 말함)
- **외형**: 하얀 비숑 프리제 (또는 지정된 동물)
- **조연**: 할머니, 동네 주민, 손님 등 (인간)
- **관계**: 동물이 인간 사회에 자연스럽게 섞여 있음
- **성격**: 당돌하고, 솔직하고, 약간 건방진 매력
- **소품**: 항상 마이크를 들고 있음 (관객에게 직접 말하는 느낌)

#### 2. 상황 설정 패턴
| 패턴 | 설명 | 예시 |
|------|------|------|
| 직업 체험 | 동물이 인간 직업을 함 | 알바, 장사, 상담사, 배달 |
| 일상 에피소드 | 동물의 평범한 하루 | 산책, 밥먹기, 낮잠 |
| 관계 드라마 | 동물과 주인/가족 갈등 | 할머니와 티격태격 |
| 사회 풍자 | 시사/트렌드를 동물 시점으로 | 보이스피싱, 물가 상승 |
| 성대모사/빙의 | 유명인/캐릭터를 동물이 흉내 | TV 속 인물 따라하기 |

#### 3. 제목 패턴 (= 펀치라인)
- 대사 중 가장 웃긴/인상적인 한 마디를 제목으로
- 10자 이내가 이상적
- 맞춤법을 일부러 틀리거나 구어체 사용 ("아니거든여", "몰라요")
- 해시태그: #웃긴영상 #귀여운영상 #귀여운강아지 #유머 #ai쇼츠

## 워크플로우

### Step 1: 상황 구상

주제를 받으면 다음을 결정한다:
- **캐릭터**: 어떤 동물이 주인공인가 (이름, 성격)
- **상황**: 무슨 일이 벌어지는가
- **갈등/반전**: 웃음 포인트가 어디인가
- **펀치라인**: 제목이 될 한 마디

### Step 2: 3단 구조 대본 작성

#### 도입 (처음 5~8초, 약 60~100자)
**목표**: 스크롤을 멈추게 하는 첫 문장

규칙:
- "안녕하세요"로 절대 시작하지 않는다
- 첫 문장에서 "뭐지?" 또는 "ㅋㅋ" 반응이 나와야 한다
- 동물이 황당한 상황에 이미 놓여 있는 채로 시작

좋은 도입 예시:
```
"아니 손님, 강아지가 왜 카운터에 앉아있냐고요?"
"오늘 알바 첫날인데 사장님이 멸치를 팔라고 하거든요."
"할머니가 오늘도 나 보고 이상한 소리를 해요."
```

나쁜 도입 예시:
```
"안녕하세요, 오늘은 강아지 이야기를 해볼게요." (X - 너무 평범)
"여러분 강아지 좋아하시죠?" (X - 질문으로 시작하면 약함)
```

#### 본문 (중간 35~42초, 약 300~350자)
**목표**: 핵심 이야기 전개 + 웃음 포인트 3개

규칙:
- 동물 1인칭 시점 유지
- 대화체: 주인공이 관객에게 직접 말하는 느낌
- 3~4개의 에피소드/비트로 구성
- 각 비트마다 웃음 포인트 또는 귀여움 포인트 하나씩
- 사투리, 줄임말 자연스럽게 사용
- 중간에 반전이나 예상 못한 전개 삽입

비트 구성:
```
비트 1: 상황 설명 + 첫 번째 웃음 (10초)
비트 2: 에피소드 심화 + 두 번째 웃음 (12초)
비트 3: 예상 밖 전개 + 세 번째 웃음 (10초)
비트 4: 클라이맥스 (10초)
```

#### 마무리 (마지막 5~8초, 약 50~80자)
**목표**: 다음 영상이 궁금해지는 한 마디

규칙:
- 열린 결말 또는 예고 느낌
- "좋아요 구독" 언급하지 않는다
- 주인공의 다음 에피소드를 암시
- 또는 철학적(?) 한 마디로 여운 남기기

좋은 마무리 예시:
```
"근데 사장님, 내일도 출근해야 하는 거에요...?"
"할머니는 모르거든요. 내가 다 들었다는 거."
"인생이 원래 그런 거래요. 그치?"
```

### Step 3: 톤 & 문체 규칙

| 항목 | 규칙 |
|------|------|
| 한 문장 길이 | 최대 25자. 짧을수록 좋다 |
| 말투 | 구어체. "~거든요", "~에요", "~잖아요" |
| 추임새 | "진짜에요", "놀랍죠?", "이거 중요해요" 적절히 삽입 |
| 시청자 호출 | "여러분은 어떠세요?", "이거 경험해보셨죠?" |
| 감정선 | 한 대본에 웃음+귀여움+약간의 감동 섞기 |
| 사투리 | 자연스럽게 섞기 ("아니거든여", "몰라유") |
| 이모지 | 사용하지 않는다 |
| 해시태그 | 제목 뒤에 5개: #웃긴영상 #귀여운영상 #귀여운강아지 #유머 #동물쇼츠 |

### Step 4: 출력 형식

다음 형식으로 대본을 출력한다:

```
제목: {펀치라인} #{해시태그1} #{해시태그2} #{해시태그3} #{해시태그4} #{해시태그5}

캐릭터: {동물 이름} ({동물 종류}, {성격 한 마디})
배경: {장소}
분위기: {mood}
예상 길이: {초}초

---

[도입] (0:00~0:0X)

{첫 문장 - 스크롤 멈추는 한 마디}
{상황 설정 1~2문장}

[본문] (0:0X~0:4X)

{비트 1}

{비트 2}

{비트 3}

{비트 4 - 클라이맥스}

[마무리] (0:4X~0:5X)

{여운 남기는 한 마디}

---

총 글자수: {N}자 (목표: 450~550자)
예상 시간: {N}초
```

### Step 5: 자체 검증

대본 작성 후 다음을 확인한다:

1. 첫 문장이 스크롤을 멈추게 하는가?
2. 한 문장이 25자를 넘지 않는가?
3. 웃음 포인트가 3개 이상인가?
4. "안녕하세요"로 시작하지 않았는가?
5. 이모지를 사용하지 않았는가?
6. 동물이 1인칭으로 말하고 있는가?
7. 총 글자수가 450~550자 범위인가?
8. 마무리가 다음 영상을 궁금하게 하는가?
9. 제목이 10자 이내의 펀치라인인가?

하나라도 실패하면 수정 후 재출력한다.
